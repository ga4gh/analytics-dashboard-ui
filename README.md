# analytics-dashboard-ui
User Interface for GA4GH Analytics Dashboard

## Configuration

The UI can be configured via the following environment variables:

| Variable Name | Description | Default |
|---------------|-------------|---------|
| `GA4GH_ANALYTICS_DASHBOARD_API_BASE_URL` | the base URL to the running dashboard API service | `http://localhost:8000` |
| `GA4GH_ANALYTICS_DASHBOARD_UI_PORT` | the port that the UI service will run on | `8050` |
